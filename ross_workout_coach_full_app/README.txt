Ross Workout Coach

Upload these files to Replit, GitHub Pages, Netlify, or Vercel:
- index.html
- style.css
- app.js
- workouts.json
- manifest.json

On iPhone:
1. Open the hosted URL in Safari.
2. Tap Share.
3. Tap Add to Home Screen.

What it does:
- One exercise per screen
- Giant suggested weight
- Completed weight input as a number only
- Swipe between exercises
- Done button auto-advances
- Saves progress locally on your phone
- Shows previous week's completed weight when the same exercise appears
- Includes 6 rowing days, 2 leg days, 3 upper/shoulder days, snatches, running progression, and seated calf raises
